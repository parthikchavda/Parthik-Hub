🔤 ટેકનિકલ શબ્દો | Technical Terms – Full Forms & Meanings (EN/GU)

📱 Activity

Full Form: N/A (component)
🇬🇧 A single screen with UI that users interact with. Manages lifecycle (onCreate, onStart, onResume...).🇮🇳 એક સ્ક્રીન જે UI ધરાવે છે, વપરાશકર્તા સાથે સંવાદ કરે છે. જીવનચક્ર મેનેજ કરે છે.

⚙️ Service

Full Form: —
Background component without UI (music playback, network tasks).પૃષ્ઠભૂમિમાં ચાલતો ઘટક, UI વગર (ગીત વગાડવું, ડાઉનલોડ).

📡 BroadcastReceiver

Full Form: —
Responds to system-wide events (battery low, SMS received).સિસ્ટમ ઇવેન્ટ્સ (બેટરી ઓછી, મેસેજ) ને પ્રતિભાવ આપે છે.

🗄️ ContentProvider

Full Form: —
Shares structured data between apps (contacts, media).એપ્સ વચ્ચે માહિતી શેર કરે છે (કોન્ટેક્ટ, ફોટા).

✉️ Intent

Full Form: —
Messaging object to request action (start activity, service).ક્રિયા માટે સંદેશ (Activity શરૂ કરવી, સેવા શરૂ).

🛠️ ADB

Full Form: Android Debug Bridge
Command-line tool to communicate with device (install, logcat).કમાન્ડ-લાઇન ટૂલ: એપ ઇન્સ્ટોલ, લોગ જોવા.

📦 APK

Full Form: Android Package Kit
Installable file format for Android apps.Android એપ્સ માટે ઇન્સ્ટોલ ફાઇલ.

📘 SDK

Full Form: Software Development Kit
Tools & libraries to build Android apps.Android એપ બનાવવા માટેનાં સાધનો.

🎭 ART / Dalvik

ART: Android Runtime | Dalvik: old VM
ART uses AOT (Ahead-Of-Time) compilation, improves performance. Dalvik used JIT.ART AOT કમ્પાઇલેશન વાપરે છે, ઝડપી. Dalvik જૂનું વર્ચ્યુઅલ મશીન.

⚡ AOT / JIT

AOT: Ahead-Of-Time | JIT: Just-In-Time
AOT compiles bytecode before execution; JIT compiles at runtime.AOT અગાઉથી કમ્પાઇલ કરે છે, JIT ચાલતી વખતે.

🗃️ SQLite / Room

SQLite: Embedded SQL DB | Room: ORM library
SQLite: lightweight relational database. Room: abstraction over SQLite with compile-time checks.SQLite: હલકો ડેટાબેઝ. Room: SQLite પર સરળ સ્તર.

📏 dp, sp, px

dp: Density-independent pixels | sp: Scale-independent pixels | px: pixels
dp for layouts (responsive), sp for text (respects font size), px absolute.dp લેઆઉટ માટે, sp ટેક્સ્ટ માટે, px ચોક્કસ પિક્સેલ.

🍞 Toast / Snackbar

—
Toast: short popup message. Snackbar: action-enabled temporary message.Toast: ટૂંકો સંદેશ. Snackbar: એક્શન સાથેનો સંદેશ.

📋 Options Menu / Context Menu

—
Options Menu (action bar), Context Menu (long press), PopupMenu (anchor view).Options Menu (ટૂલબાર), Context Menu (લાંબું દબાવો), PopupMenu.

💬 AlertDialog

—
Modal dialog to confirm, show messages or choices. Builder pattern.પુષ્ટિ માટેનું ડાયલોગ બોક્સ.

♻️ RecyclerView

—
Advanced list with ViewHolder pattern, LayoutManager, efficient recycling.મોટી યાદી માટે આધુનિક ઘટક, ViewHolder અને રિસાયક્લિંગ.

🧩 Fragment

—
Reusable UI portion inside Activity, has own lifecycle.Activity ની અંદર પુનઃઉપયોગી UI ભાગ.

🔌 HAL

Hardware Abstraction Layer
Interface between Android framework and hardware drivers.હાર્ડવેર અને ફ્રેમવર્ક વચ્ચે પુલ.

📡 IPC / Binder

Inter-Process Communication
Binder is Android's IPC mechanism for cross-process calls.પ્રોસેસ વચ્ચે સંચાર માટે Binder સિસ્ટમ.

🎨 Material Design

—
Google's design system (elevation, shadows, responsive motion).Google ની ડિઝાઇન ભાષા (છાયા, એનિમેશન).

🔄 Activity Lifecycle Methods – Full meanings (EN + GU)

onCreate()
🇬🇧 First call, initialize UI
🇮🇳 પ્રથમ કૉલ, UI બનાવો
onStart()
Activity visible, not interactive
દૃશ્યમાન, હજુ સંપર્ક નથી
onResume()
User interacts (foreground)
યુઝર સાથે સંવાદ
onPause()
Partially hidden, save data
અંશતઃ છુપાયેલ, ડેટા સાચવો
onStop()
Completely hidden, release resources
સંપૂર્ણ છુપાયેલ, સંસાધનો છોડો
onRestart()
Coming back from stopped state
બંધ સ્થિતિમાંથી પાછા
onDestroy()
Final cleanup
અંતિમ સફાઈ

Example: onStop() → release camera, stop animations. onResume() → restart animations. Full forms: No acronyms, but key terms explained bilingually.
