⚡ કન્વર્ઝન ફોર્મ્યુલા
Decimal → Binary (÷2 ની પ્રક્રિયા)
13₁₀ = 1101₂ (13÷2=6R1, 6÷2=3R0, 3÷2=1R1, 1÷2=0R1 → ઉલ્ટો વાંચો)
Binary → Decimal (ઘાત પ્રક્રિયા)
1101₂ = 1×2³+1×2²+0×2¹+1×2⁰ = 8+4+0+1 = 13₁₀
Hex → Binary (દરેક Hex = 4 Bits)
A₁₆ = 1010₂, F₁₆ = 1111₂, 2B₁₆ = 0010 1011₂
BCD, Gray Code, Excess-3
• BCD: Binary Coded Decimal – 0-9 ને 4 bits માં દર્શાવો
• Gray Code: Adjacent numbers – ફક્ত 1 bit change (Error ↓)
• Excess-3: BCD + 3 = Self-complementing code
🧠 યાદ રાખો: "Binary Decimal Gray Excess" = BDGE

🔥 De Morgan's – EXAM में સૌથી વધારે
"AND ના complement = OR of complements"
(AB)' = A' + B' | Real Use: NAND Gate = NOT-AND
K-Map (Karnaugh Map) – SOP Minimization
• 2-variable: 2×2 grid (4 cells)
• 3-variable: 2×4 grid (8 cells) – Gray code order
• 4-variable: 4×4 grid (16 cells)
• Grouping Rule: 1, 2, 4, 8, 16 (2ⁿ) – Adjacent cells
✓ Tip: Wrapping (Corners + Edges) ભૂલશો નહીં! Largest group = best simplification
SOP (Sum of Products) vs POS (Product of Sums)
SOP: F = AB + A'C | POS: F = (A+B)(A'+C)
🧠 K-Map Trick: "Ones Group = SOP | Zeros Group = POS"

Counters (કાઉન્ટર)
• Ripple/Async: FF cascaded – Slow but simple
• Sync Counter: Clock connected to all FFs – Fast
• Up Counter: 0→1→2→...→Max→0 (MOD-N)
• Down Counter: Max→...→0→Max
n-bit counter = 2ⁿ states | MOD-10 = BCD counter
4-bit: 0000 to 1111 (0 to 15) = MOD-16
🧠 "JK = Jack of All Trades, D = Data Direct, T = Toggle"

🔄 ADC / DAC Conversion + PLA (Programmable Logic Array)
DAC (Digital → Analog)
• R-2R Ladder: Simple, accurate, popular
• Weighted Resistor: 2R,4R,8R... for each bit
V_out = V_ref × (D/2ⁿ)
• Resolution: 1/2ⁿ × Full Scale | 8-bit = 256 levels
• % Resolution: (1/2ⁿ) × 100
DAC Key Terms
Resolution: ઓછામાં ઓછો voltage step
Settling Time: Output stable થવાનો time
Accuracy: Actual vs Ideal output difference
ADC (Analog → Digital)
• Flash ADC: Fastest! 2ⁿ-1 comparators, Parallel
• SAR ADC: Successive Approx Register – Medium speed
• Dual Slope: Most Accurate, Slow (DVMs)
• Sigma-Delta: High resolution audio/sensor
Nyquist Rate: f_s ≥ 2 × f_max
• Quantization Error: ±½ LSB max
ADC Key Terms
LSB = Least Significant Bit step size
Conversion Time = ADC ને digital output આવવા સુધી
PLA – Programmable Logic Array
• PLA: Programmable AND + OR array
• PAL: Fixed OR + Programmable AND (faster)
• ROM as PLA: Fixed AND + Programmable OR
PLA Structure
Inputs → AND Array (Programmable) → OR Array (Programmable) → Outputs
• n inputs × p product terms × m outputs
📌 PLA vs PAL vs ROM
PLA: Both P+P | PAL: P-AND + Fixed-OR | ROM: Fixed-AND + P-OR

CMOS Inverter (સૌથી Important!)
CMOS Inverter Structure
VDD → pMOS → Output → nMOS → GND | Input → Both Gates
• Input=0: pMOS ON, nMOS OFF → Output=VDD (1)
• Input=1: pMOS OFF, nMOS ON → Output=GND (0)
• Static Power: 0 (No DC path!) | Dynamic Power: C×V²×f
P_dynamic = α × C_L × V_DD² × f_clk
MOS Regions of Operation
• Cutoff: V_gs < V_th → I_D = 0 (OFF)
• Linear/Triode: V_ds < V_gs-V_th → Switch ON
• Saturation: V_ds ≥ V_gs-V_th → Amplifier region
📌 V_th Threshold Voltage
V_th = minimum gate voltage to create channel | nMOS: +ve | pMOS: -ve
🔮 FPGA & VHDL
FPGA – Field Programmable Gate Array
FPGA Internal Structure
CLB (Configurable Logic Block) + I/O Blocks + Routing Channels + RAM Blocks + DSP
• LUT (Look-Up Table): FPGA નો basic element – RAM-based
• CLB: LUT + Flip-Flop + MUX combination
• Reprogrammable: SRAM-based → Can change design!
• Anti-fuse: One-time programmable FPGA
• Vendors: Xilinx, Altera (Intel), Lattice
VHDL – Very High Speed IC Hardware Description
entity AND_gate is
  port(A,B: in std_logic;
      Y: out std_logic);
end AND_gate;
architecture RTL of AND_gate is
begin Y <= A and B; end;
• Entity: Port declarations (interface)
• Architecture: Behavior/RTL/Structural
• Signal vs Variable: Signal=Global | Variable=Local
• Concurrent: <= | Sequential: := (in Process)
📝 🔥 Frequently Asked MCQs & Previous Year Questions (PSI / GPSC / GSSSB)
Q1. NAND gate ને Universal Gate કેમ કહે?
✓ Ans: NAND gate વડે AND, OR, NOT – ત્રણે gates બનાવી શકાય
Q2. 4-bit Binary Counter ની Modulus?
✓ Ans: MOD-16 (2⁴=16 states: 0000 to 1111)
Q3. De Morgan's Theorem: (A+B)' = ?
✓ Ans: A' · B' (NOR = AND of complements)
Q4. Flash ADC ની વિશેષતા?
✓ Ans: Fastest ADC | 2ⁿ-1 comparators parallel
Q5. CMOS Logic Family ના ફાયદા?
✓ Ans: Very low static power dissipation | Wide voltage range (3-15V)
Q6. SR Flip-Flop Invalid State?
✓ Ans: S=1, R=1 → Forbidden/Invalid state
Q7. JK FF J=1, K=1 → ?
✓ Ans: Toggle (Q → Q' – Output complement)
Q8. Nyquist Sampling Theorem?
✓ Ans: f_s ≥ 2×f_max (min sampling = 2×highest freq)
Q9. K-Map group size 4 = ?
✓ Ans: 2 variables eliminate (n cells = n/2 variables eliminated)
Q10. VLSI ક્ષેત્ર – Moore's Law?
✓ Ans: Transistor count doubles every 18-24 months
Q11. DRAM vs SRAM?
✓ DRAM: Capacitor (slow,cheap,needs refresh) | SRAM: FF (fast,costly,no refresh)
Q12. FPGA ના Basic Element?
✓ Ans: LUT (Look-Up Table) – basic programmable cell

📌 Important Definitions
• Fan-out: Maximum number of gates a gate output can drive
• Propagation Delay: Time for output to respond to input change
• Setup Time: Data must be stable before clock edge
• Hold Time: Data must remain stable after clock edge
• Metastability: FF output oscillates when setup/hold violated
